# ACME-Shopping

This project use Java Spring Boot framwork, Angular 10 to build the back-end, front-end and use MongoDB as the database.

Back-End:
Java Spring Boot,
Entity Model,
Api controller,
Resource,
Repository

Front-End:
Shopping Component,
Shopping Cart Component,
Header Component,
Footer Component,
Email Receipt Feature

Database:
model 
{
    "_id": number,
    "name": string,
    "summary": string,
    "image": string,
    "price": number,
    "stock": number,
    "_class": string
}
